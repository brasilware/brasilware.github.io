Introdução

O Brasilware é uma plataforma completa para análise de malware, projetada para ajudar profissionais de segurança cibernética a identificar, analisar e documentar ameaças de maneira eficiente. A plataforma possui uma arquitetura modular que permite estender suas funcionalidades conforme necessário.

O que você pode fazer com o Brasilware:

  • Instalar e executar módulos especializados de análise
  • Combinar múltiplos relatórios em um único documento
  • Regenerar relatórios com análise de IA
  • Personalizar a interface conforme suas preferências

Gerenciamento de Módulos

Os módulos são componentes fundamentais do Brasilware que adicionam funcionalidades específicas à plataforma.

Instalação de Módulos

1

Acesse a Página de Módulos

Clique em "Módulos" na barra de navegação.

2

Clique na "página de módulos "

Isso abrirá o formulário de instalação.

3

Preencha as Informações

Informe o nome e a descrição do módulo.

4

Selecione o Método de Instalação

Escolha entre fazer upload de um arquivo ou especificar uma URL.

  • Arquivo: Clique em "Selecionar arquivo" e escolha o arquivo HTML do módulo.
  • URL: Digite a URL completa do módulo.
5

Clique em "Instalar módulo"

Aguarde até que o módulo seja instalado com sucesso.

Execução de Módulos

Para executar um módulo instalado:

  1. Acesse a seção "Módulos"
  2. Localize o módulo desejado na lista de módulos instalados
  3. Clique no botão "Executar"
  4. O módulo será aberto em uma SandBox e você poderá interagir com sua interface

Remoção de Módulos

Para remover um módulo:

  1. Acesse a seção "Módulos"
  2. Localize o módulo na lista de módulos instalados
  3. Clique no botão "Remover"
  4. Confirme a remoção na caixa de diálogo de confirmação

Fusão de Relatórios

A funcionalidade de fusão de relatórios permite combinar múltiplos documentos PDF em um único arquivo.

Como Fusionar PDFs

1

Acesse a Seção de Fusão

Clique em "Fusionar Relatórios" na barra de navegação.

2

Adicione os PDFs

Clique no botão "Adicionar PDFs" e selecione os arquivos que deseja combinar. Você pode selecionar vários arquivos de uma vez.

3

Organize os PDFs

Use os botões de seta para cima e para baixo para reorganizar a ordem dos PDFs conforme necessário.

4

Inicie a Fusão

Clique no botão "Fusionar PDFs Selecionados" para iniciar o processo de fusão.

5

Baixe o Resultado

Após a conclusão, clique em "Baixar PDF" para salvar o arquivo combinado.

Histórico de Mesclagens

Todos os PDFs mesclados anteriormente aparecem na seção "Histórico de Mesclagens" na parte inferior da página. Você pode:

  • Baixar novamente um PDF mesclado anteriormente
  • Regenerar o PDF com IA (requer configuração de API)
  • Remover PDFs que não são mais necessários

Integração com IA

O Brasilware oferece recursos avançados de análise de malware através da integração com inteligência artificial.

Importante:

Para utilizar a funcionalidade de IA, é necessário configurar uma chave de API válida nas configurações do sistema.

Configuração da API

  1. Acesse a seção "Configurações"
  2. Role até a seção "Configurações de API"
  3. Insira sua chave de API no campo correspondente
  4. Clique em "Salvar" para aplicar as alterações

Regeneração de Relatórios com IA

Para regenerar um relatório usando IA:

  1. Vá para a seção "Fusionar Relatórios"
  2. Localize o PDF desejado no histórico de mesclagens
  3. Clique no botão "Regenerar com IA"
  4. Selecione o idioma desejado para o relatório regenerado
  5. Clique em "Regenerar relatório"
  6. Aguarde a conclusão do processo
  7. Baixe o novo relatório aprimorado pela IA

Configurações

A seção de configurações permite personalizar diversos aspectos da plataforma Brasilware.

Tema

O Brasilware oferece tema claro e escuro. Para alterá-lo:

  1. Acesse a seção "Configurações"
  2. Na seção "Tema", selecione o modo desejado (Claro ou Escuro)
  3. As alterações são aplicadas imediatamente

Idioma

A plataforma suporta múltiplos idiomas. Para alterar o idioma da interface:

  1. Acesse a seção "Configurações"
  2. Na seção "Idioma", selecione o idioma desejado no menu suspenso (Português, Inglês ou Espanhol)
  3. Ele salva automaticamente e altera

Chave de API

Para configurar a integração com IA:

  1. Acesse a seção "Configurações"
  2. Role até a seção "Configurações de API"
  3. Insira sua chave de API no campo correspondente
  4. Clique em "Salvar" para aplicar as alterações

Solução de Problemas

Problemas Comuns e Soluções

Módulo não carrega

Solução: Verifique se o módulo está em formato HTML válido. Tente reinstalar o módulo ou carregar a partir de uma fonte diferente.

Erro ao mesclar PDFs

Solução: Certifique-se de que todos os PDFs selecionados ainda existem no sistema. Verifique se os arquivos não estão corrompidos e tente novamente.

Erro na regeneração com IA

Solução: Verifique se a chave de API foi configurada corretamente nas configurações. Certifique-se de que a chave tem saldo suficiente e permissões adequadas.

Interface não responde

Solução: Atualize a página ou limpe o cache do navegador. Se o problema persistir, reinicie o navegador ou servidor.
