Introducción

Brasilware es una plataforma completa para el análisis de malware, diseñada para ayudar a los profesionales de ciberseguridad a identificar, analizar y documentar amenazas de manera eficiente. La plataforma tiene una arquitectura modular que permite ampliar su funcionalidad según sea necesario.

Lo que puedes hacer con Brasilware:

  • Instalar y ejecutar módulos de análisis especializados
  • Combinar múltiples informes en un solo documento
  • Generar informes con análisis mediante IA
  • Personalizar la interfaz de acuerdo a tus preferencias

Gestión de módulos

Los módulos son componentes fundamentales de Brasilware que añaden funcionalidades específicas a la plataforma.

Instalación de módulos

1

Accede a la página de módulos

Haz clic en "Módulos" en la barra de navegación.

2

Haz clic en "Página de módulo"

Esto abrirá el formulario de instalación.

3

Completa la información

Ingresa el nombre y la descripción del módulo.

4

Selecciona el método de instalación

Elige entre cargar un archivo o especificar una URL.

  • Archivo: Haz clic en "Seleccionar archivo" y elige el archivo HTML del módulo.
  • URL: Ingresa la URL completa del módulo.
5

Haz clic en "Instalar módulo"

Espera hasta que el módulo se instale correctamente.

Ejecución de módulos

Para ejecutar un módulo instalado:

  1. Accede a la sección "Módulos".
  2. Localiza el módulo deseado en la lista de módulos instalados.
  3. Haz clic en el botón "Ejecutar".
  4. El módulo se abrirá en un entorno seguro y podrás interactuar con él.

Eliminación de módulos

Para eliminar un módulo:

  1. Accede a la sección "Módulos".
  2. Localiza el módulo en la lista de instalados.
  3. Haz clic en el botón "Eliminar".
  4. Confirma la eliminación en el cuadro de diálogo de confirmación.

Informes

La funcionalidad de informes permite combinar múltiples documentos PDF en un solo archivo.

Cómo fusionar PDFs

1

Accede a la sección de fusión

Haz clic en "Fusionar informes" en la barra de navegación.

2

Agrega los PDFs

Haz clic en el botón "Agregar PDFs" y selecciona los archivos que deseas combinar. Puedes seleccionar múltiples archivos a la vez.

3

Organiza los PDFs

Utiliza los botones de flecha arriba y abajo para reorganizar el orden de los PDFs según sea necesario.

4

Inicia la fusión

Haz clic en el botón "Fusionar PDFs" para comenzar el proceso de fusión.

5

Descarga el resultado

Una vez finalizado, haz clic en "Descargar PDF" para guardar el archivo combinado.

Historial de mensajes

Todos los PDFs fusionados aparecen en la sección "Historial de mensajes" al final de la página. Desde allí, puedes:

  • Descargar un PDF fusionado previamente
  • Regenerar el PDF con IA (previa configuración de la API)
  • Eliminar PDFs que ya no sean necesarios

Integración con IA

Brasilware ofrece funciones avanzadas de análisis de malware mediante tecnología de inteligencia artificial.

Importante:

Para utilizar la funcionalidad de IA, debes configurar una clave API válida en las configuraciones del sistema.

Configuración de API

  1. Accede a la sección "Configuración".
  2. Desplázate a la sección "Configuración de API".
  3. Ingresa tu clave API en el campo correspondiente.
  4. Haz clic en "Guardar" para aplicar los cambios.

Regeneración de informes con IA

Para regenerar un informe utilizando IA:

  1. Accede a la sección "Fusionar informes".
  2. Localiza el PDF deseado en el historial de fusión.
  3. Haz clic en "Regenerar con IA".
  4. Selecciona el idioma deseado para el informe regenerado.
  5. Haz clic en "Regenerar informe".
  6. Espera a que se complete el proceso.
  7. Descarga el nuevo informe potenciado por IA.

Configuración

La sección de configuración te permite personalizar varios aspectos de la plataforma Brasilware.

Tema

Brasilware ofrece temas claro y oscuro. Para cambiarlo:

  1. Accede a la sección "Configuración".
  2. En la sección "Tema", selecciona el modo deseado (claro u oscuro).
  3. Los cambios se aplican de inmediato.

Idioma

La plataforma soporta múltiples idiomas. Para cambiar el idioma de la interfaz:

  1. Accede a la sección "Configuración".
  2. En la sección "Idioma", selecciona el idioma deseado del menú desplegable (Portugués, Inglés o Español).
  3. El cambio se guarda automáticamente.

Clave API

Para configurar la integración con IA:

  1. Accede a la sección "Configuración".
  2. Desplázate a la sección "Configuración de API".
  3. Ingresa tu clave API en el campo correspondiente.
  4. Haz clic en "Guardar" para aplicar los cambios.

Solución de problemas

Problemas comunes y soluciones

El módulo no se carga

Solución: Verifica que el módulo esté en un formato HTML válido. Intenta reinstalar el módulo o cargarlo desde otra fuente.

Error al fusionar PDFs

Solución: Asegúrate de que todos los PDFs seleccionados sigan existiendo en el sistema. Verifica que los archivos no estén corruptos y vuelve a intentarlo.

Error en la regeneración con IA

Solución: Comprueba que la clave API esté configurada correctamente en las configuraciones y que tenga saldo suficiente y los permisos adecuados.

La interfaz no responde

Solución: Actualiza la página o limpia la caché del navegador. Si el problema persiste, reinicia el navegador o el servidor.

Soporte técnico

Si encuentras problemas que no puedes solucionar, contacta al soporte técnico:

  • Abrir un problema en Github
  • Ver el repositorio en Github

Consejo para la solución de problemas:

Al reportar un problema, incluye la siguiente información:

  • Versión de Brasilware
  • Navegador y versión utilizados
  • Descripción detallada del problema
  • Pasos para reproducir el problema
  • Capturas de pantalla, si es posible
