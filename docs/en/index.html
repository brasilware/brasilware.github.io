Key Features

  • Modular Analysis: Install and run different analysis modules to detect and classify malware.
  • Report Merging: Combine multiple reports into a single document.
  • AI Regeneration: Enhance existing reports using artificial intelligence.
  • Multilingual Support: Interface available in Portuguese, English, and Spanish.

Getting Started Guide

  1. Set up the environment following the deployment instructions.
  2. Install analysis modules from the library or create your own.
  3. Upload reports for analysis and processing.
  4. Use AI-powered features to enhance your existing reports.

Frequently Asked Questions

How can I contribute to the project?

Contributions are welcome! Visit our GitHub repository to learn how to contribute with code, translations, or documentation.

What are the minimum system requirements?

To run Brasilware in a production environment, we recommend at least 1GB RAM, 1 CPU core, and 20MB of available storage.
